\noindent{\textbf{Resumen}\\
	

Se pretende desarrollar una aplicación web para la Policía Local que permita a los agentes del cuerpo conseguir un mejor aprovechamiento del tiempo y realizar su trabajo
de manera más fácil. Debido a que la gran mayoría de las acciones cotidianas de un policía local se hacen de manera manual y rutinaria, se pretende crear un sistema que ayude 
al agente a realizar este proceso de manera automática, rápida y sencilla desde cualquier sitio.

\noindent{\textbf{Abstract}\\


The main objective is to develop a web application for the local police that allows the officers to have a better time managment and to perform their work easier. Because the vast majority of the daily actions 
of a local police officer are done manually and routinely, this application creates a system that helps the officer to perform this tasks automatically, quickly and easily
from anywhere.
